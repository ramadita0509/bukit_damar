# Website Informasi Bukit Damar

Website informasi resmi tentang Bukit Damar - destinasi wisata alam yang menawarkan keindahan panorama dan pengalaman yang tak terlupakan.

## Tentang Bukit Damar

Bukit Damar adalah destinasi wisata alam yang menawarkan pemandangan indah dan udara segar. Terletak di kawasan yang asri, Bukit Damar menjadi pilihan tepat untuk melepas penat dan menikmati keindahan alam.

## Fitur Website

Website ini menyediakan berbagai informasi tentang Bukit Damar, termasuk:

- **Informasi Umum** - Sejarah dan profil Bukit Damar
- **Fasilitas** - Fasilitas yang tersedia untuk pengunjung
- **Galeri** - Koleksi foto keindahan Bukit Damar
- **Lokasi & Akses** - Informasi cara menuju Bukit Damar
- **Kontak** - Informasi kontak untuk pertanyaan lebih lanjut

## Teknologi

Website ini dibangun menggunakan:

- **Laravel** - Framework PHP untuk backend
- **Tailwind CSS** - Framework CSS untuk styling
- **Vite** - Build tool untuk assets

## Instalasi

1. Clone repository ini
2. Install dependencies:
   ```bash
   composer install
   npm install
   ```
3. Copy file `.env.example` menjadi `.env`:
   ```bash
   cp .env.example .env
   ```
4. Generate application key:
   ```bash
   php artisan key:generate
   ```
5. Setup database di file `.env`
6. Jalankan migrasi:
   ```bash
   php artisan migrate
   ```
7. Build assets:
   ```bash
   npm run build
   ```
8. Jalankan development server:
   ```bash
   composer run dev
   ```
